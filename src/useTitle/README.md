# @nooks/use-title

React Hook to update your document's title.

## Installation

**yarn**

yarn add @nooks/use-title

**npm**

npm i @nooks/use-title

## Usage

```react
import React from "react";
import useTitle from "@nooks/use-title";

function App() {
  useTitle("Welcome");
  return <h1>Welcome</h1>;
}
```
